TikTok callback

Redirect URI support page for the ShortsFlow TikTok web app configuration.

Registered redirect page

This static page is hosted on the same public domain as the ShortsFlow website and can be registered as the TikTok web redirect URI for app review.

How ShortsFlow uses it

The production ShortsFlow Hub keeps TikTok credentials server-side. When TikTok redirects an authorized account back to this domain, the authorization result is handled by the operator workflow and is not exposed on this public review page.

Reviewer note

If an error parameter appears in the URL, authorization did not complete and the app configuration, redirect URI, or requested scopes should be reviewed in TikTok for Developers. No token or credential should be pasted into this public page.
