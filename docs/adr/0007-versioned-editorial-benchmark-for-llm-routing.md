# Versioned editorial benchmark for LLM routing

ShortsFlow will compare text models through a **Benchmark Editorial Versionado** instead of isolated smoke tests. This keeps the same **Conjunto de Benchmark Editorial**, **Perfis de Evidencia** and **Cartoes de Evidencia** across providers, so **Torneio de LLMs** results can drive **Roteamento Editorial por Etapa** using cost, speed, factual obedience and editorial quality rather than impressions from one-off jobs.
