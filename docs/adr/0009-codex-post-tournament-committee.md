# Codex post-tournament committee

ShortsFlow will not use an automatic LLM judge for every candidate output in the **Torneio de LLMs** because that made the benchmark slow and token-expensive. The tournament will generate compact, standardized artifacts, apply objective local gates first, and use Codex as a **Comite de Decisao Pos-Torneio** only on viable finalists to recommend cost-benefit and routing without erasing the **Ranking por Etapa do Torneio**.
