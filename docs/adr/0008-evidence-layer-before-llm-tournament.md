# Evidence layer before LLM tournament

ShortsFlow will harden the **Camada de Evidencia** before using full pipeline runs to rank text models. One-off provider tests can validate credentials and JSON compatibility, but provider quality decisions should wait until **Perfis de Evidencia**, **Fontes de Evidencia**, **Cartoes de Evidencia** and the **Benchmark Editorial Versionado** prevent tangential sources from contaminating the comparison.
