Creator publishing operations system

ShortsFlow

ShortsFlow is the public review surface for the same operational Hub used to create, review, schedule, and publish approved short-form videos to YouTube and TikTok.

Watch TikTok demo Open integration demo OAuth redirect page

The sandbox review demo shows OAuth context, approved video package, TikTok publication queue, upload request, and status tracking.

Authorized creators only

TikTok publishing runs only after the TikTok account owner grants access.

Human review first

Script, visual, audio, render, monetization, and review gates run before automated posting.

Operational audit trail

The Hub records scheduling state, TikTok publish IDs, status checks, and errors.

TikTok Content Posting API

End-to-end integration flow

01Connect

The authorized TikTok account grants the required publishing scope through OAuth.

02Approve

The operator reviews the generated short, metadata, captions, audio, and quality gates.

03Publish

ShortsFlow creates a TikTok publication record and submits the approved video package.

04Track

The worker syncs TikTok publication status and stores the outcome in the Hub.

Data processed

Access token, creator info, generated video file, caption, hashtags, publish ID, status, and API errors.

Data purpose

Used only to publish approved videos, check status, avoid duplicates, and show operational state.

No unrelated sharing

ShortsFlow does not sell, rent, or use TikTok data for advertising.
