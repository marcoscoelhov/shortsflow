Private creator publishing system

ShortsFlow

ShortsFlow is the public identity for the ShortsFlow review and publishing workflow. It helps one authorized channel operator create, review, schedule, and publish approved short-form videos to YouTube and TikTok.

Watch TikTok demo Open live mockup OAuth redirect page

The sandbox/mockup review demos show the complete TikTok Content Posting API flow: OAuth context, approved video package, TikTok publication queue, upload request, and status tracking.

Authorized account only

TikTok publishing runs only after the TikTok account owner grants access. ShortsFlow does not offer public user registration.

Human review first

Videos pass through script, visual, audio, render, monetization, and review gates before any automated TikTok posting job is eligible.

Operational audit trail

The hub records scheduling state, TikTok publish identifiers, status checks, and errors so duplicate posting can be avoided.

TikTok Content Posting API

End-to-end integration flow

01

Connect

The authorized TikTok account grants the required publishing scope through TikTok OAuth.

02

Approve

The operator reviews the generated short, metadata, captions, audio, and quality gates.

03

Publish

ShortsFlow creates a TikTok publication record and submits the approved video package.

04

Track

The worker syncs TikTok publication status and stores the outcome in the private hub.

Data processed

Access token, creator info, generated video file, caption, hashtags, publish identifier, publication status, and TikTok API errors.

Data purpose

The data is used only to publish approved videos, check status, avoid duplicate posts, and show operational state to the operator.

No third-party marketplace

ShortsFlow does not manage unrelated creator accounts and does not sell, rent, or use TikTok data for advertising.
